## 🚀 Getting Started

### Prerequisites

- Node.js `>=20` (see note below if you're on Node 20)
- Yarn `>=1.22`

### Install Dependencies

```bash
yarn install
```

> ⚠️ **Node Version Note**
>
> Some dependencies (e.g. `@babel/generator@8.x`) require Node `>=22.18.0` or `>=24.11.0`.
> If you're on **Node 20**, run:
>
> ```bash
> yarn install --ignore-engines
> ```
>
> This bypasses the engine check and installs all packages successfully.

### Start Development Server

```bash
yarn dev
```

App runs at → [http://localhost:5173](http://localhost:5173)
